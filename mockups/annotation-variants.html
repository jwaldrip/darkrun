darkrun — one surface: view = annotate

There's no separate "review" screen. You open an artifact; the annotation palette is always there; pick a tool anytime. Three artifact classes, same envelope, each resolving to a source target the agent can act on. The green card under each is literally what the agent receives.

1   Text — code / markdown / spec
Select a span → comment, highlight, strike, or propose a replacement (stored as a diff). Anchored by version_sha + line range + the quoted text & surrounding context, so it survives the file moving.
